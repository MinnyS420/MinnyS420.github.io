Tic-Tac-Toe

This is my project with JavaScript.


This code is for a tic-tac-toe game. The board is represented by an array of nine empty strings. The currentPlayer variable stores the player's turn
(X or O). The squares variable stores all the squares in the game board. The message variable stores the message that will be displayed to the user.
